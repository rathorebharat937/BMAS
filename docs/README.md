# BMAS Project Documentation

This folder contains the system documentation, database schemas, API specs, and project reports for the Business Management & Analysis System (BMAS).

## Contents
- **System Architecture**: High-level and component-level designs.
- **Database Diagrams**: Entity-Relationship Diagrams (ERD) and schemas.
- **API Documentation**: Endpoint specs and request/response examples.
- **User Manuals**: Guidelines for administrators, managers, and employees.
